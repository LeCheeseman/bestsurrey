# Best Surrey Content Growth Pipeline

Last updated: 2026-07-08

This is the recommended operating pipeline for growing Best Surrey without adding lots of weak pages or generic AI content.

## Goal

Build search visibility by improving pages Google is already testing, keeping the site visibly updated, and turning cleaned listing data into useful editorial content.

The order matters:

```text
Clean listings -> strengthen hub pages -> publish useful guides -> monitor Search Console -> repeat
```

Do not start by generating a large number of blog posts. The directory data has to become trustworthy first.

## Weekly Workflow

### 1. Pull Search Console Data

Export Search Console every 1 to 2 weeks:

```text
Performance -> Search results -> Export
```

Review:

```text
Pages with high impressions and low position
Queries with impressions around positions 8-30
Pages with impressions but zero clicks
Old URLs still appearing
Pages gaining impressions after recent edits
```

Priority rule:

```text
Position 8-20 = quick-win page
Position 20-50 with high impressions = build page quality and internal links
Position 50+ = usually not ready unless impressions are growing
```

### 2. Pick One Page Cluster

Work in clusters, not random individual listings.

Examples from the July 2026 Search Console export:

```text
Woking brunch
Farnham restaurants / places to eat / pubs
Guildford things to do
Surrey brunch
```

For each cluster, define the target page:

```text
/woking/cafes-brunch
/farnham
/farnham/restaurants
/farnham/pubs-bars
/guildford/things-to-do
/cafes-brunch
```

### 3. Research the SERP

For the chosen query group, search manually:

```text
best brunch in Woking
things to do in Guildford
places to eat Farnham
best pubs in Farnham
```

Record:

```text
Who ranks now
What intent they satisfy
What listings they include
What Best Surrey is missing
Whether the winning pages are directories, official tourism pages, blogs, Tripadvisor, local publishers, or venue pages
```

The goal is not to copy competitors. The goal is to understand what Google thinks the query means.

### 4. Clean Listings in Admin

Use:

```text
/admin/category-review
/admin/listing-qa
```

For the target cluster:

```text
Remove bad or closed listings
Merge duplicates
Fix wrong categories
Add missing obvious venues
Check website, phone and address
Add 3 to 5 strong images per important listing
Rewrite short summaries
Rewrite long descriptions where thin
Mark reviewed only when complete
```

Important: do not enrich photos for listings that may be removed.

### 5. Strengthen the Hub Page

For the target page, improve:

```text
Title tag
Meta description
Intro
Internal links to subcategories
Internal links to related towns/categories
Listing order
Listing coverage
Photo quality
```

The page should answer the search intent quickly. For example:

```text
/woking/cafes-brunch
  -> brunch, breakfast, coffee shops, bakeries

/guildford/things-to-do
  -> walks, gardens, historic sites, indoor entertainment, family days out

/farnham/restaurants
  -> places to eat, date night, family dining, Sunday roast, pub food overlap
```

### 6. Decide Whether a Blog/Guide Is Justified

Only create a blog-style guide when it adds something the directory page cannot.

Good guide ideas:

```text
A relaxed Saturday in Guildford: walk, lunch and afternoon stop
Best rainy-day things to do in Guildford
Where to go for brunch in Woking this weekend
Farnham pub walk: traditional pubs and nearby green spaces
Family day out in Surrey: soft play, lunch and an easy park
```

Weak guide ideas:

```text
Top 10 restaurants in Surrey
Best things to do in Surrey
Best cafes in Surrey
```

Those broad topics are competitive and often duplicate existing category pages.

## Listing Research Pipeline

Use this pipeline for adding or improving listings.

### Input

```text
Target town
Target category/subcategory
Search Console queries
Competitor SERP review
Known local must-have venues
```

### Research Checks

For each candidate venue:

```text
Does it exist?
Is it independent or genuinely useful?
Is it already in Best Surrey?
Which main categories should it belong to?
Which subcategories should it belong to?
Does it have a good official website?
Does it have official photos?
Is there a menu, booking page, ticket page or family information page?
```

### Admin Integration

In admin:

```text
Add or update the listing
Assign main category/categories
Assign subcategories
Add website, phone and address
Upload images
Write short summary
Write long description
Add highlights
Set price/family-friendly where useful
Mark reviewed
```

### Quality Bar

A listing is not complete unless it has:

```text
Accurate name
Correct town
Correct categories
Working official website where available
Phone or clear contact route where available
Address or location
At least 3 good photos for priority listings
Useful short summary
Non-generic long description
No obvious duplicate
```

## AI-Assisted Content Pipeline

AI can help, but it must not invent facts.

### Safe AI Uses

```text
Summarising official venue information
Drafting short summaries from verified notes
Suggesting internal links
Suggesting blog ideas from Search Console clusters
Creating first drafts from a structured brief
Checking descriptions for generic wording
Creating title/meta variants
```

### Unsafe AI Uses

```text
Inventing reviews
Inventing opening hours
Inventing menu items
Inventing claims like "award-winning"
Writing confident descriptions from only a venue name
Producing large batches of unreviewed pages
```

### AI Brief Template

Use this structure for drafts:

```text
Page/listing:
Target query:
Search intent:
Verified facts:
Official website:
Current Best Surrey listing URL:
Related internal links:
Must avoid:
Tone:
Output needed:
```

Example:

```text
Page/listing: Woking brunch guide
Target query: brunch Woking
Search intent: find good local cafes/breakfast spots
Verified facts: list of cleaned Woking brunch listings from admin
Official website: use only URLs attached to listings
Related internal links: /woking/coffee-shops, /woking/brunch-spots, /cafes-brunch
Must avoid: invented opening hours, Google user photo claims
Tone: useful, local, concise
Output needed: intro, 4 internal-link blurbs, 6 listing summary improvements
```

## Blog/Guide Publishing Pipeline

### 1. Generate Ideas From Data

Inputs:

```text
Search Console query export
Pages gaining impressions
Cleaned listing clusters
Seasonality
Local events/holidays
```

Idea rules:

```text
Each guide must link to at least 5 cleaned listings
Each guide must support an existing category/town page
Each guide must answer a specific intent
Each guide must be reviewed before publishing
```

### 2. Build a Guide Brief

Each guide should include:

```text
Target query
Secondary queries
Audience
Suggested title
Internal links
Listings to include
Image needs
Manual checks required
```

### 3. Draft With AI

Draft from the brief only. Use AI to create:

```text
Title
Meta description
Intro
Route plan or sections
Listing blurbs
Internal links
Image captions
```

### 4. Human Review

Before publishing:

```text
Check every fact
Check every listing still exists
Check every internal link
Check image rights/source
Remove generic filler
Make sure the guide adds value beyond the category page
```

### 5. Publish and Link Back

After publishing:

```text
Link guide from related town/category pages
Link relevant listings back to the guide if appropriate
Add guide to sitemap only if it is public and strong
Monitor impressions after 2 to 4 weeks
```

## Monthly Review

Each month:

```text
Export Search Console
List top 20 pages by impressions
List top 20 queries by impressions
Identify pages moving from positions 20-50 to 8-20
Identify pages with high impressions but stale/no clicks
Pick 3 clusters for the next month
Archive or redirect weak/stale URLs
```
